# 🍽️ RecipeRender

**RecipeRender** is a modern, intuitive web application that allows food enthusiasts to discover, create, and manage their favorite recipes. Built with **React**, it offers a seamless and responsive user experience with a clean and elegant design.

---

## 🚀 Live Demo

https://reciperender.netlify.app/

---

## 📖 About the Project

RecipeRender simplifies the way you organize your culinary creations. Whether you're a home cook or a professional chef, this app provides a beautifully designed interface to browse, add, and edit recipes effortlessly.

---

## ✨ Features

- 🔍 **Recipe Browsing** – Explore recipes in beautiful card layouts with detailed views.
- 🗂️ **Category Filtering** – Filter recipes by category (Breakfast, Lunch, Dinner, etc.).
- ✍️ **Create & Edit Recipes** – Add your own creations using a simple and intuitive form.
- 📱 **Responsive Design** – Enjoy a smooth experience on desktops, tablets, and phones.
- ⚡ **Real-Time Updates** – Instantly reflect changes using React state management.

---

## 🛠️ Tech Stack

| Tech             | Description                                      |
|------------------|--------------------------------------------------|
| **React**        | Component-based UI development                   |
| **React Router** | Client-side routing for seamless navigation      |
| **Context API**  | Global state management for recipes              |
| **React Hook Form** | Form management with validation              |
| **Tailwind CSS** | Utility-first CSS with a dark theme              |
| **Vite**         | Lightning-fast dev server and build tool         |
